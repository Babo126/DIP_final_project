# DIP 期末專題 - 互動式影像處理工具

這是一個基於 Python Tkinter 與 OpenCV 開發的視窗化影像處理應用程式。
本專案採用物件導向（OOP）架構設計，專注於人像處理，提供精準的 **膚色檢測與校正** 以及趣味性的 **角色風格濾鏡** 功能。

## 功能特色

### 1. 核心基礎功能
* **直覺的 GUI 介面**：包含左側工具列、右側預覽窗格與中央畫布。
* **響應式縮放**：中央畫布與右側預覽圖會隨視窗大小自動等比例縮放，不變形。
* **完整操作歷史**：支援 **復原 (Undo)** 與 **取消復原 (Redo)**，方便回溯操作步驟。
* **檔案存取**：支援讀取與儲存常見圖片格式 (JPG, PNG)。

### 2. 進階影像處理 (DIP Algorithms)
* **膚色校正 (Skin Tone Correction)**：
    * 使用 **HSV 色彩空間** (`[0, 40, 70]` ~ `[25, 255, 255]`) 進行精準的膚色遮罩 (Masking) 提取。
    * 提供 **亮度 (Brightness)**、**對比度 (Contrast)**、**Gamma 校正** 與 **色調 (Hue)** 的即時滑桿調整。
    * **非破壞性預覽**：調整過程中僅顯示預覽，按下「套用」才正式寫入。

* **角色濾鏡 (Character Filters)**：
    * 內建多種風格濾鏡：**哥布林 (Goblin)**、**納美人 (Na'vi)**、**小小兵 (Minion)**、**派大星 (Patrick)**。
    * 自動針對膚色區域進行色相偏移與紋理合成。
    * **紋理處理**：結合高斯雜訊 (Gaussian Noise) 與模糊 (Blur) 技術模擬皮膚質感。

## 技術棧

* **語言**：Python 3.x
* **GUI 框架**：Tkinter (內建)
* **影像處理**：OpenCV (`cv2`), NumPy
* **影像顯示**：Pillow (`PIL`)
